# Privacy Policy

**Last updated:** February 28, 2026

## Overview

Prism is an automated content creation and affiliate marketing tool for personal use. This application uses the Pinterest API to publish pins on behalf of its operator.

## Data Collection

Prism does not collect, store, or share personal data of Pinterest users. The application only accesses data that is explicitly authorized by the account owner through Pinterest OAuth.

## Data Usage

- Pinterest account credentials (OAuth tokens) are used solely to publish pins on behalf of the authorized account owner.
- No data is shared with third parties.
- No analytics or tracking data is collected from Pinterest users.

## Third-Party Services

This application integrates with the Pinterest API in accordance with [Pinterest's Developer Terms of Service](https://developers.pinterest.com/terms/).
