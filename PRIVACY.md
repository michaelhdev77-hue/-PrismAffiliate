# Privacy Policy

**Last updated:** March 2, 2026

## Overview

Prism is a personal content management tool that helps individual creators plan, schedule, and publish pins on Pinterest. The application provides a review and approval workflow, giving the account owner full control over every pin before it is published.

## Data Collection

Prism does not collect, store, or share personal data of Pinterest users. The application only accesses data explicitly authorized by the account owner through Pinterest OAuth2.

Specifically, Prism accesses:
- Pinterest account profile (to verify connected account)
- Boards list (to let the owner select a destination board)
- Pin creation and read permissions (to publish and retrieve pins created by this app only)

## Data Usage

- Pinterest OAuth tokens are stored securely and used solely to perform actions explicitly approved by the account owner.
- All pins go through a manual review step before any API call is made — nothing is published without the owner's approval.
- No data is sold, shared, or used for advertising.
- No data from Pinterest users other than the account owner is accessed or stored.

## Data Retention

OAuth tokens are stored in an encrypted form and can be revoked at any time via Pinterest account settings or within the application. Revocation immediately terminates all API access.

## Third-Party Services

Prism integrates with the Pinterest API in accordance with the [Pinterest Developer Guidelines](https://policy.pinterest.com/en/developer-guidelines) and [Pinterest Terms of Service](https://developers.pinterest.com/terms/).
